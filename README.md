# AI Skills Library

A centralized library of AI skill files for use across n8n workflows via HTTP Request nodes.

## How to Use in n8n

Fetch any skill via HTTP Request node:
```
GET https://raw.githubusercontent.com/gnuspeak/ai-skills-library/main/{category}/{skill-name}.md
```
Then inject the response body into your Anthropic node's System Prompt field.
